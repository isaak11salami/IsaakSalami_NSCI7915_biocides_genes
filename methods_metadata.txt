Methods of Data Collection: This study utilises the "Source Figure 1." dataset obtained from https://www.nature.com/articles/s41564-023-01474-z#Sec27 on 20250323. It is associated with the paper, "Systematic analyses identify modes of action of ten clinically relevant biocides and antibiotic antagonism in Acinetobacter baumannii" (Li et al., 2023) DOI: https://doi.org/10.1038/s41564-023-01474-z. 
For information on the methods used to generate the data displayed in this dataset, please refer to Methods in the original publication.

Data processing: All data processing and statistical analysis will be conducted in R Studio (R Core Team (2025). _R: A Language and Environment for Statistical Computing_. R Foundation for Statistical Computing, Vienna, Austria. <https://www.R-project.org/>.)

Data will be analysed using R's inbuilt factor analysis function. Plots generated in the final report will also be generated using R's inbuilt plot function.

Independent Variable abbreviation key: The biocides assessed in the original study are tabulated in the dataset using an abbreviated code. Below are the full names of each abbreviated biocide "level"
BZK: benzalkonium
CTAB: cetryltrimethylammonium bromide
CHL: chlorhexidine
TRC: triclosan
CRL: chloroxylenol
PVPi: polyvidone iodine
HClO: sodium hypochlorite
GLU: glutaraldehyde
EtOH: ethanol
